## How to use

- Clone or download zip
- Use ``` cd frontend ``` and just open ``` index.html ``` in a browser
- To start developing: ``` cd frontend ``` , ``` compass watch ```

```sh
 1. git clone https://github.com/SerjoWeb/site-constructor.git
 2. cd frontend
 3. compass watch
```
